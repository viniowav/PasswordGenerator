# Password Generator

A simple password generator.

# Features

- User can choose the password length
- Generates powerful and secure passwords:
  - Uppercase characters (A-Z)
  - Lowercase characters (a-z)
  - Numbers (0-9)
  - Special characters (!@#$%^&*)
  
  User can choose whether to use or not numbers or special characters.
    
- Fast and simple
